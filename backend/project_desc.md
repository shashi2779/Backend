### Database : mongoDb,Mongoose
### Authentication : Json web token
### 3rd party : 
         * Payment Gateway : RazorPay
         * Email : nodemailer,gmail
### Testing : PostMan
### Frontend : React , Backend : Express
### Deployment :
    * Backend : Heroku
    * Frontend : Netlify
    * Codebase : Github
    * Database server : mongoDb atlas
